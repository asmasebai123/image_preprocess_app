# image_preprocess_app# ImageLab Pro - Plateforme de Prétraitement d'Images

Application web complète pour le prétraitement d'images avec interface utilisateur moderne et fonctionnalités avancées.

## Fonctionnalités

### 1. Ajustements de base
- Luminosité
- Contraste
- Niveaux de gris
- Teinte

### 2. Filtres
- Flou gaussien
- Flou moyen
- Flou médian
- Filtre bilatéral
- Motion blur
- Accentuation

### 3. Transformations
- Rotation (-90°, +90°, angle personnalisé)
- Miroir horizontal/vertical
- Redimensionnement
- Recadrage

### 4. Seuillage
- Seuillage binaire
- Seuillage adaptatif
- Méthode d'Otsu

### 5. Séparation des canaux
- Canal rouge
- Canal vert
- Canal bleu
- Tous les canaux

### 6. Détection de contours
- Canny
- Sobel
- Laplacien

### 7. Histogramme
- Visualisation RGB
- Canaux séparés
- Égalisation d'histogramme

### 8. Exportation
- Formats PNG, JPG, TIFF
- Contrôle de qualité
- Téléchargement direct
